# Análisis de Ventas de Amazon (2022–2023)

## Descripción del proyecto
Este proyecto consiste en el análisis de las ventas de Amazon durante los años 2022 y 2023 utilizando Microsoft Excel.  
Se ha llevado a cabo un proceso completo de limpieza de datos, análisis exploratorio y visualización mediante un dashboard interactivo.

## Objetivos
- Limpiar y preparar los datos originales para su análisis.
- Crear nuevas variables que faciliten el estudio de la información.
- Analizar los ingresos según el tiempo, la categoría, la región y el método de pago.
- Evaluar el impacto de los descuentos en las ventas.
- Comparar los resultados obtenidos entre los años 2022 y 2023.
- Presentar los resultados a través de un dashboard interactivo.

## Origen de los datos
Los datos utilizados en este proyecto provienen de la plataforma Kaggle.

Enlace al dataset original:  
https://www.kaggle.com/datasets/aliiihussain/amazon-sales-dataset

Los datos originales se incluyen en este repositorio en formato CSV, dentro de la carpeta `data`.

## Metodología
1. Importación de los datos originales en Excel.
2. Limpieza y transformación de los datos (formatos, duplicados y variables).
3. Creación de nuevas variables:
   - Año
   - Mes
   - Trimestre
   - Pedido con descuento / sin descuento
4. Análisis exploratorio mediante tablas dinámicas.
5. Cálculo de indicadores clave (KPIs).
6. Creación de gráficos y dashboard interactivo con segmentadores.

Todo el proceso de análisis se encuentra desarrollado en un único archivo de Excel.

## Herramientas utilizadas
- Microsoft Excel (fórmulas, tablas dinámicas, gráficos y segmentadores).
